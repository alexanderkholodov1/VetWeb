# Guia para pasar esta documentacion a Notion

## Opcion 1: copiar y pegar (recomendada)
1. Crear pagina nueva en Notion: VetWeb - Documentacion.
2. Abrir docs/NOTION_VETWEB_DOCUMENTACION.md.
3. Copiar todo y pegar en Notion.
4. Repetir con docs/CHECKLIST_ENTREGA.md.

## Opcion 2: importacion markdown
1. En Notion, click en Import.
2. Elegir Markdown & CSV.
3. Importar carpeta docs o cada archivo markdown.

## Estructura sugerida en Notion
- 01. Resumen
- 02. Arquitectura
- 03. Roles y Reglas
- 04. Despliegue
- 05. Operacion
- 06. Checklist
- 07. Pendientes
